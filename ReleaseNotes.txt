Update Version 1.0.12 Features:
- Integrated the Teacher Dashboard directly into the main window as a new Tab for seamless navigation.
- Added Data Zoom In/Out capability (Ctrl + Mouse Wheel) to the Dashboard data tables.
- Implemented an "Unmatched Files" feature to track and view print jobs with malformed or missing teacher/level names.
- Enhanced print log matching logic to more accurately handle different file naming conventions.
- Added dynamic Date Columns to the Teacher Dashboard to view pages printed per day.
- Added 5 Weeks Monthly Auto-Generation and Grading logic.
- Added "Teacher Schedule Settings" allowing customization of teaching and printing days (e.g., exemptions for non-teaching days).
- Improved Dashboard UI layout responsiveness (WrapPanel) and visual clarity (font colors for labels).
- Minor UI layout fixes and performance improvements.
