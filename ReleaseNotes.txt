Update Version 1.0.23 Features:
- Fixed external CSV saving bug: Editing an external CSV file and saving now writes back to its original path instead of overwriting today's default CSV print log.
- Synchronized Dashboard updating: Data modifications (edits, deletions, cancels, pastes, and undo/redo actions) now immediately trigger a dashboard recalculation.
- Fixed Windows Startup capturing bug: When the application starts minimized/hidden on Windows boot (via the -background argument), all print spooler tracking and SNMP polling services now initialize immediately.
- Optimized Tab switching performance: Offloaded all heavy calculations (exemption matching, grade evaluation, and LCS name matching) to a background thread and implemented DataGrid UI control caching/reuse to achieve instantaneous tab switching with zero lag.
- Restored print detail hover tooltips on teacher names.

Update Version 1.0.22 Features:
- Improved Auto Print error handling: added immediate skip to next available files on printing error without blocking the queue.
- Implemented intelligent retry mechanism: failed files are automatically retried after all unfailed files in the queue have been processed.
- Added 3-retry limit with 1-minute cooldown per file: after 3 consecutive failures, a specific file enters a 60-second cooldown while allowing other files to print normally without stopping Auto Print.
- Fixed Teacher Dashboard external CSV data synchronization so external logs appear consistently on both Print Tracker and Dashboard tabs without merging into default live data.
- Enhanced teacher schedule matching algorithm with strict Level verification and normalization (handling variations like Pre2A1, Pre-2A-I) to prevent mismatched assignment across grade levels.
